# Run Tests

1. Run the full test suite: `npm test`
2. If specific files were changed, run only related tests: `npm test -- --testPathPattern="<pattern>"`
3. Report results clearly — list passing and failing tests
4. If tests fail, read the failure output and suggest fixes
